# Overview

Information about the project:

Essentially, this program is a kinematic-to-dynamic telemetry calculator.\
It gathers data from user input (pace note and speed), then the data is translated (note to radius (m), and speed to velocity (m/s)). Then, lateral Gs, weight transfer (N) and suspension travel (mm) is calculated. Finally, the telemetry output will display the approximate raw numbers!
